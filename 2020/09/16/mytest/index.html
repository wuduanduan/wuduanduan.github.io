命令备忘

Posted on 2020-09-16 Edited on 2020-09-17

Macbook

  • 进入对用户隐藏的folder，比如~/Library/Caches/: Cmd+Shift+G;
  • 一次修改该类型文件的打开方式：选中文件，右键，GetInfo，bar下方open with选中App，changeAll。

Jupyter notebook

  • 如果将ipynb导出为易读文件，html比较好，若内有plotly图表:
    jupyter nbconvert --to HTML --execute nbname.ipynb

Git

  • 查看Git是否关联到本地: ssh -T git@github.com

Hexo

  • 修改页面字体大小:themes/next/source/css/_variables/base.styl
